# Freelance Music API

Minimal Express + TypeScript API for lesson scheduling demo.

- `GET /health`
- `POST /api/teachers`
- `GET /api/teachers`
- `POST /api/students`
- `GET /api/students`
- `POST /api/bookings`
- `GET /api/bookings`
- `GET /api/admin/reports/bookings`
- `POST /api/payments/validate-card` { cardNumber }

Scripts:
- `npm run dev`
- `npm run build` then `npm start`
